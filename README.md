# Ceiling Hub Website

A modern product showcase website built for **Ceiling Hub Pvt. Ltd.** to display ceiling systems, insulation materials, and construction boards.
The website allows users to explore products, view detailed product pages, and request quotes directly via WhatsApp.

---

## Live Features

* Product listing with modern card layout
* Individual product detail pages
* Image gallery for each product
* WhatsApp quote integration
* Customer support page
* Contact form
* Responsive design for mobile and desktop
* Animated UI using Framer Motion
* Professional navigation and layout

---

## Tech Stack

* **React**
* **Vite**
* **Tailwind CSS**
* **Framer Motion**
* **React Router**
* **React Icons**

## Installation

Clone the repository

```
git clone https://github.com/mad2media2025-afk/ceiling-hub.git
```

Navigate into the project

```
cd ceiling-hub
```

Install dependencies

```
npm install
```

Run development server

```
npm run dev
```

The project will run on:

```
http://localhost:5173
```

---

## Build for Production

```
npm run build
```

Preview production build

```
npm run preview
```
